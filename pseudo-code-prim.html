ConstruireArbreARPM() :
marquer tous les sommets comme non visités
pour chaque sommet du graphe boucler
    VisiterLeSommet()
fin boucler

VisiterLeSommet() :
si le sommet est non visité alors
    déposer le sommet et une priorité quelconque quand la queue de priorité
    tant que la queue de priorité n'est pas vide boucler
        prélever le sommet de tête (il devient le sommet courant)
        marquer le sommet courant comme visité 
        traiter le sommet courant
        pour chaque sommet S voisin du sommet courant, boucler
            si S n'est pas visité alors
                calculer la priorité de S (poids de l'arc)
                si S n'est pas rencontré alors
                    déposer S avec sa priorité -> queue de priorité
                sinon si la nouvelle priorité est plus basse alors
                    diminuer la priorité de S dans la file de priorité
                fin si
            fin si
        fin boucler
    fin boucler
fin si
